# 认领一项 API

## 大纲

API 可以是很好玩的。这里有[许多公开 API 的清单](https://github.com/public-apis/public-apis)。挑选一项 API，建立一个网页扩充功能来解决问题。问题可以很小，如找不到足够的宠物照片，这时你可以尝试使用[dog CEO API](https://dog.ceo/dog-api/))；或是解决更大问题。好好享受吧！
