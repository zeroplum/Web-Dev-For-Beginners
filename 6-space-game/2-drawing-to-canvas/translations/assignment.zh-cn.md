# 把玩 Canvas API

## 简介

挑选一款 Canvas API 上的元素，为它建立一些有趣的设定。你能利用重复的星星建立银河吗？你能建立有特殊材质的线条吗？你可以观察 CodePen 上的范本激发想法，但请不要抄袭。
